# Realtime Django Chat

A real-time chat application built with Django, RabbitMQ, Django Channels, and Vue.js. The project demonstrates how to build a full-stack messaging system with authenticated users, real-time notifications, message delivery, and a modern frontend client.

## Tech Stack

### Backend

* Python
* Django 2.0
* Django Channels
* Djoser
* Django Notifs
* Pika
* RabbitMQ
* uWSGI

### Frontend

* Vue.js
* Vue CLI
* Vue Router

## Features

* User authentication
* Real-time chat messaging
* WebSocket-based communication
* RabbitMQ message broker integration
* Notification system
* Vue.js single-page frontend
* REST API integration between frontend and backend
